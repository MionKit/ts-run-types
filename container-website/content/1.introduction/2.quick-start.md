## 1. Install

You need two things: the runtime package and the Vite plugin that reads your types.

::code-group
```bash [pnpm]
pnpm add ts-runtypes
pnpm add -D vite-plugin-runtypes
```
```bash [npm]
npm install ts-runtypes
npm install -D vite-plugin-runtypes
```
```bash [yarn]
yarn add ts-runtypes
yarn add -D vite-plugin-runtypes
```
::

::note
RunTypes is the only thing that ends up in your runtime bundle, and it carries no dependencies. `vite-plugin-runtypes` does its work at build time and ships nothing to production.
::
