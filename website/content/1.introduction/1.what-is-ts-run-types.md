# What is ts-run-types

<!-- skeleton — full copy lands in the content phase -->

The one-paragraph pitch: TypeScript erases your types and calls it a feature. **ts-run-types** reads them at build time and hands you back validators, JSON + binary (de)serializers, mock-data generators and runtime reflection — no schemas, no drift, no writing your types twice.
