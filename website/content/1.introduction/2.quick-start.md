# Quick Start

Zero to a working `createValidate<User>()` in about three minutes. Two packages, two config lines, then write a type and call it.

## 1. Install

You need two things: the runtime package and the Vite plugin that reads your types.

::code-group
```bash [pnpm]
pnpm add @mionjs/ts-go-run-types
pnpm add -D vite-plugin-runtypes
```
```bash [npm]
npm install @mionjs/ts-go-run-types
npm install -D vite-plugin-runtypes
```
```bash [yarn]
yarn add @mionjs/ts-go-run-types
yarn add -D vite-plugin-runtypes
```
::

::note
`@mionjs/ts-go-run-types` is the only thing that ends up in your runtime bundle — and it carries no dependencies. `vite-plugin-runtypes` does its work at build time and ships nothing to production.
::

## 3. Turn on reflection in tsconfig

The plugin needs the type information, so flip on `reflection` in your `tsconfig.json`:

```json [tsconfig.json]
{
  "compilerOptions": {
    "reflection": true
  }
}
```

::tip
Forget this and the build can't see your types — `createValidate<T>()` won't have anything to generate from. It's the one easy-to-miss step.
::
