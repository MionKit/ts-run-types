## 1. Install

You need two things: the runtime package and the Vite plugin that reads your types.

::code-group
```bash [pnpm]
pnpm add @mionjs/ts-go-run-types
pnpm add -D vite-plugin-runtypes
```
```bash [npm]
npm install @mionjs/ts-go-run-types
npm install -D vite-plugin-runtypes
```
```bash [yarn]
yarn add @mionjs/ts-go-run-types
yarn add -D vite-plugin-runtypes
```
::

::note
`@mionjs/ts-go-run-types` is the only thing that ends up in your runtime bundle — and it carries no dependencies. `vite-plugin-runtypes` does its work at build time and ships nothing to production.
::
