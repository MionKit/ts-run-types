## Type functions — `createX`

Each factory takes the type as a generic (`createValidate<User>()`) or infers it from a value, and returns a specialized function generated at build time. All of them validate / serialize the **serializable** projection of `T` (functions, methods and symbols are dropped — see [the validate contract](/reference/diagnostics)).

| Export | Returns | What it does |
|--------|---------|--------------|
| `createValidate<T>()` | `ValidateFn<T>` | Fast type guard — `(v) => v is DataOnly<T>`. |
| `createGetValidationErrors<T>()` | `GetValidationErrorsFn` | Detailed error report — array of `RunTypeError`. |
| `createJsonEncoder<T>()` | `JsonEncoderFn<T>` | Value → JSON-ready (`Date`/`BigInt`/`Map`/`Set` survive). Strategies: `clone` (default), `mutate`, `direct`. |
| `createJsonDecoder<T>()` | `JsonDecoderFn<DataOnly<T>>` | JSON-parsed value → restored types. Strategy: `strip` (default) / `preserve`. |
| `createBinaryEncoder<T>()` | `BinaryEncoderFn<T>` | Value → compact `Uint8Array`. |
| `createBinaryDecoder<T>()` | `BinaryDecoderFn<DataOnly<T>>` | Binary buffer → restored types. |
| `createMockType<T>()` | `MockTypeFn<T>` | Generates valid, type-shaped mock data. |
| `createHasUnknownKeys<T>()` | `HasUnknownKeysFn` | `true` if the value carries keys not in `T`. |
| `createStripUnknownKeys<T>()` | `StripUnknownKeysFn` | Deletes undeclared keys in place, returns same ref. |
| `createUnknownKeyErrors<T>()` | `UnknownKeyErrorsFn` | One `{path, expected:'never'}` per undeclared key. |
| `createUnknownKeysToUndefined<T>()` | `UnknownKeysToUndefinedFn` | Sets undeclared keys to `undefined`. |
| `createFormatTransform<T>()` | `FormatTransformFn` | Applies type-format transforms (trim, case, …). |

## Schema builders — `RT.*`

From `@mionjs/ts-go-run-types/schema`. Each returns a `RunType<…>` node; `typeof builder(...)` *is* the run-type and `Static<typeof …>` recovers the TypeScript type.

**Leaf / atomic** — `string` · `number` · `boolean` · `bigint` · `literal` · `regexp` · `symbol` · `any` · `unknown` · `never` · `void` · `classType` · `enum` (`enumType`) · `brand(name)`

**Date / time** — `date` · `temporal.*` (the 8 TC39 `Temporal` types)

**Composers** — `object` · `array` · `tuple` · `union` · `intersection` · `record` · `map` · `set` · `promise` · `func` · `callable` · `templateLiteral` · `circular((self) => …)` / `self()` · `propMod({optional?, readonly?}, field)` · `optional(field)`

**Utility-type builders** — `partial` · `required` · `pick` · `omit` · `exclude` · `extract` · `nonNullable` · `readonly` (`readonlyType`) · `returnType` · `parameters`

**String formats** — `alpha` · `alphaNumeric` · `numeric` · `lowercase` · `uppercase` · `capitalize` · `uuidv4` · `uuidv7` · `ip` · `ipv4` · `ipv6` · `ipWithPort` · `ipv4WithPort` · `ipv6WithPort` · `domain` · `domainUnicode` · `domainPunycode` · `domainStrict` · `email` · `emailPunycode` · `emailStrict` · `url` · `urlHttp` · `urlFile` · `stringDate` · `stringTime` · `stringDateTime`

**Number formats** — `integer` · `float` · `positive` · `negative` · `positiveInt` · `negativeInt` · `int8` · `int16` · `int32` · `uint8` · `uint16` · `uint32`

**BigInt formats** — `bigPositive` · `bigNegative` · `bigPositiveInt` · `bigNegativeInt` · `bigInt64` · `bigUInt64`

## Markers

The type-level brands the build scanner recognizes at call sites. All but `getRunTypeId`/`reflectRunTypeId` are types only.

| Export | Kind | Meaning |
|--------|------|---------|
| `getRunTypeId<T>()` | fn | Stable type id, static form (you supply `T`). |
| `reflectRunTypeId(value)` | fn | Stable type id, reflection form (`T` inferred from the value). |
| `InjectRunTypeId<T>` | type | Trailing-slot marker — declare it to wrap the reflection helpers. |
| `InjectTypeFnArgs<T, Fn>` | type | Trailing-slot marker for the `createX` factory family. |
| `CompTimeArgs<T>` | type | Brands an argument as "must be a literal (or const-of-literal)". |
| `CompTimeFnArgs<T>` | type | `CompTimeArgs` whose literal value also selects the `createX` variant. |
| `PureFunction<F>` | type | Brands an argument as "inline function that passes the purity rules". |

## Registries & helpers

| Export | What it does |
|--------|--------------|
| `registerClassSerializer(name, {serialize, deserialize})` | Custom (de)serializer for a user class (JSON + binary). |
| `registerPureFnFactory(ns, fn, factory)` | Register a pure function the AOT compiler can inline. |
| `registerMockingFunction(kind, fn)` | Custom mock generator for a type-format kind. |
| `registerFormatPattern(pattern)` | Register a regex-based string-format pattern. |
| `getRTUtils()` / `getRTFnCaches()` | Access the runtime RT registry / function caches. |
| `createDataViewSerializer(key, opts?)` | Pre-build a binary serializer for buffer reuse. |
| `createDataViewDeserializer(...)` | Pre-build a binary deserializer for buffer reuse. |
| `setSerializationOptions(opts)` | Tune binary buffer sizing globally. |
| `RunTypeKind` | Enum mirror of reflection kinds (for custom formats). |

## Types

| Type | What it is |
|------|------------|
| `ValidateOptions` | `{noLiterals?, noIsArrayCheck?}` — `createValidate` / `createGetValidationErrors` options. |
| `JsonEncoderOptions` | `{strategy?: 'clone' \| 'mutate' \| 'direct'}`. |
| `JsonDecoderOptions` | `{strategy?: 'strip' \| 'preserve'}`. |
| `BinaryEncoderOptions` / `BinaryDecoderOptions` | `{cacheKey?}` — buffer-history bucketing. |
| `HasUnknownKeysOptions` | `{checkNonRTProps?}`. |
| `MockOptions` / `RunTypeMockOptions` | Bounds + pools for `createMockType`. |
| `RunTypeError` | `{path: RunTypeErrorPathSegment[], …}` — one validation error. |
| `RunTypeErrorPathSegment` | One segment of a validation-error path. |
| `TypeFormatError` | A format-level validation failure. |
| `DataOnly<T>` | The serializable projection of `T` (decoders return this). |
| `Static<RT>` | Recovers the TS type a `RunType` carries. |
| `RunType<T>` | The generic run-type node every `RT.*` builder returns. |
| `ValidateFn` / `GetValidationErrorsFn` | Return types of the validate factories. |
| `JsonEncoderFn` / `JsonDecoderFn` | Return types of the JSON factories. |
| `BinaryEncoderFn` / `BinaryDecoderFn` / `ToBinaryFn` / `FromBinaryFn` | Binary fn types. |
| `MockTypeFn` | Return type of `createMockType`. |
| `TypeFormat` / `TypeFormatBase` / `TypeFormatParams` | Type-format authoring types. |
| `FormatAnnotation` / `FormatPattern` / `StringPatternArgs` | Format metadata types. |
| `ClassSerializer` / `MockFormatFn` | Registry callback types. |
| `RTUtils` | The runtime RT registry shape. |
| `DataViewSerializer` / `DataViewDeserializer` | Pre-built binary (de)serializer instances. |
| `SerializationOptions` / `CreateSerializerOptions` | DataView tuning options. |
| `StrictArrayBuffer` / `BinaryInput` | Binary I/O input types. |
| `PropModifiers` / `MapTuple` / `TemplatePart` / `AssembleTemplate` / `BrandArg` | `RT.*` builder helper types. |
| `RunTypeKindValue` | The value type of the `RunTypeKind` enum. |
