## Footnotes

::note{title="вљ пёЏ Partial support"}
- **`symbol`** validates structurally, but symbols can't be serialized вЂ” at a property position they are silently dropped (VL013 / per-family `*013`); at a root or propagating position a serializer emits an Error and throws (VL002, PJ005, вЂ¦). See [Diagnostics](/reference/diagnostics).
- **`Uppercase` / `Lowercase` / `Capitalize`** are recognized at the type level but the string *transform* is not applied at runtime yet вЂ” the value is validated as a plain string.
- **Conditional & mapped types** resolve through tsgo before we ever see them, so a conditional/mapped type that resolves to a concrete shape works fine. What's "partial" is the open-ended case where the type stays generic at the call site.
::

::note{title="рџљЂ Optimized discriminated unions"}
When a union has a literal discriminant property, ts-run-types compiles a single switch on the tag instead of trying each member in order вЂ” the same fast path the runtime uses for serialization.
::

::note{title="рџ’Ћ Things only ts-run-types does (from this table)"}
- **`Date`, `RegExp`, `Map`, `Set`, and the eight TC39 `Temporal.*` types** are first-class: validated, JSON round-tripped (they survive the wire), binary-encoded, and mocked. No other library in the table covers `Temporal`.
- **Function reflection** вЂ” `ReturnType<F>`, `Parameters<F>`, and reflecting a function value directly (`reflectRunTypeId(fn)`). Typia is type-first too but cannot extract types *from* a function signature; the rest are schema-first and can't see functions at all. This is the single biggest differentiator вЂ” see the [comparison](/reference/comparison#the-function-reflection-edge).
::

## A note on `any` and unions

`any` is permitted everywhere, but it's almost never what you want: `createValidate<any>()` is a noop that accepts every value, and serializers fall back to `JSON.stringify` / `JSON.parse` (slower, and a build-time diagnostic fires). Unions can't contain `any` or `unknown` вЂ” they match everything at runtime, which makes union discrimination impossible.
